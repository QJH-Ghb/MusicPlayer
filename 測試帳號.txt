請改用 User Secrets：

dotnet user-secrets set "Authentication:Google:ClientId" "你的ClientID"
dotnet user-secrets set "Authentication:Google:ClientSecret" "你的ClientSecret"


這樣 appsettings.json 可以保持乾淨，而程式仍會自動讀到。

範例：Google 登入回傳 Claims
{
  "sub": "11324792391827364",
  "name": "Ryo Qiu",
  "given_name": "Ryo",
  "family_name": "Qiu",
  "picture": "https://lh3.googleusercontent.com/a-/AOh14Gj...",
  "email": "ryo@example.com",
  "email_verified": true,
  "locale": "zh-TW"
}
